This page should help you if you are experiencing problems with the election site.

The most fiddly part of submitting your vote in an election is probably the part where you need to enter the code from the nationstates site. I have tried to make this as simple as possible but it can still sometimes be a little confusing. Essentially, you need to do the following:

  1. Click the "get code" button on the voting page. Note that on some devices this button is sometimes hard to see; it is to the right of the "enter code" box.
  2. Copy the code in the box from the NationStates site that pops up. Notes:
    • You need to be logged in on NationStates as the nation you are voting as. (This is the nation you chose above the get code box).
    • This code will change every time you vote, so there is no point in writing it down anywhere
  3. Close the popup box
  4. Paste the code you copied earlier into the box that says "enter code here".
  5. Fill out the rest of the form, and click submit.
If you are still having problems after this, please contact your administrator.
